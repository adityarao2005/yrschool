Topic 4 - Radicals

A radical is represented by the symbol √. In this lesson, we will be dicussing when the radical represents a square root.
The answer to a square root is the number that, when multiplied by itself, gives the number under the square root.
For example, √4 gives 2, because 2*2=4.

When we are multiplying two radicals, eg. √a * √b, we can combine them into 1, as √ab.
This means that we can also pull them apart, by doing √ab=√a * √b.
This allows us to do simplifications like √20 = √4 * √5 = 2*√5.

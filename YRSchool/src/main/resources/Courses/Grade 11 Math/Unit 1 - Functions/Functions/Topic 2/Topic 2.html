Topic 2 - Function Notation

A function f(x) is another way of representing a graph/relation where y = something.
However, in order for a relation to be a function, it must pass something called the vertical line test.

For any function, it is needed to guarantee that each value of x corresponds to exactly one value of y.
In other words, each input can only correspond to one output.

Example 1: Is f(x)= x2 a function?

In order to determine this, we can graph this function, and see if it passes the vertical line test.
The graph of f(x) is shown below.


















To do the vertical line test, imagine drawing a straight, vertical line down the graph, at any value of x.
If this line touches multiple parts of the graph, then we can say that the relation fails the vertical line test.
If there is no vertical line that can be drawn that touches multiple parts of the graph, then we say that the relation passes the vertical line function.
In this case, the relation passes the vertical line test. This means that it is a function.

An example of a relation that fails the vertical line test is y2=x, the graph of which is shown below.


















As can be seen, a vertical line can be drawn at x=5, that passes through the function at 2 points.
This means that this relation is not a function, hence why it was denoted as y2=x instead of using function notation.
